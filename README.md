
# Wireframe p/ computação gráfica

Projeto visando construir um visualizador 3D na ferramenta Qt para disciplina de Computação Gráfica. 
Objetivos: criar as próprias estruturas de desenho, usando minimamente funções nativas das bibliotecas.

Entrega 01:

- [ ] Criar display file para: pontos, linhas e polígonos

- [ ] Visualizar objetos do display file no frame

- [ ] Objetos devem ser representados por coordenada de mundo
