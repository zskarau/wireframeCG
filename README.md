
# 3D Wireframe Visualizer

This is a 3D wireframe visualizer project done for the Computer Graphics university class using QT as a form of understanding in depth the fundamental concepts of graphic rendering in computers. 

Thus, the application was done using only QT native functions, using the minimal as posible of external libraries.

The main implementation goals of the project are to:

- [ ] Create a display file and display its contents on screen

- [ ] Implement Transformations using matrices

- [ ] Include Viewport

- [ ] Read .obj files and display them on screen

- [ ] User control of transformations
